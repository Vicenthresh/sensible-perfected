# Vicenthresh's Sensible MC - V2

A vanilla-enhanced Minecraft modpack for 1.21.1 (Fabric).

Built for friends who want Minecraft to feel better without feeling different.

## Install

1. Download [PrismLauncher](https://prismlauncher.org/)
2. Add Instance > Modrinth > Search "Vicenthresh's Sensible MC" (once published)
3. Launch and play

## What this pack does

- Makes the game run better (Sodium, Lithium, etc.)
- Makes the world more interesting to explore (better structures, biomes, atmosphere)
- Adds quality-of-life without changing core gameplay
- Improves combat for those who want it
- Expands building options
- Supports multiplayer with voice chat and shared maps

## What this pack does NOT do

- Add tech mods or automation
- Add magic systems or RPG progression
- Make the game unrecognizable
- Require a wiki to play
